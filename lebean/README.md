# Le Bean

A warm and wholesome reflected XSS fun for the whole family!

## How?

Yes, yes. I'm getting there.

```
python3 -m venv venv/
source venv/bin/activate
pip3 install -r lebean/requirements.txt
cd lebean/
flask run
```

Open `http://localhost:5000` in browser, and have fun!
